Box Styles, or using CSS to create a page layout

Figuring out the dimensions of each component on an HTML page can be difficult. Margins, padding, floats, widths, heights, there's a lot to consider. In the tutorial below we provide a simple process which will get the basic layout of any HTML page off the ground.

Creating Your Page Layout

Open the code editor here

Step 1: Build your basic HTML structure as outlined above. Or use the generated structure the code editor offers by clicking the link above.

Step 2: The page layout shown in the image is based on the outline above. It features all of the typical sections a home page would have. Logo, navbar, header, your main content area, a headline box, and columns below. This tutorial will not be responsive, but will show the basic use of floats, margins, and padding and how they create a page structure such as this.

Step 3: Copy the page structure shown in the image. The entire page is contained without a div named "container". Take note of the fact that the header tags contain the logo and nav divs. The main div also contains the headline div. This is where floats come into play.

Step 4: First we will start with the container, header and the elements contained within it. We are using an external CSS file, if you are not familiar with this, check out the tutorial here.

Step 5: Create the selectors for the header, logo, and nav:



Step 6: Next we will create the main and headline selectors. Note that main has a float: left, and headline has a float: right. This keeps each piece in its corresponding position on the left and right. By adjusting the margins we are able to align them as seen in the outline.

Step 7: Next we need to create the columns: Notice each of them float left, this is because it allows us to keep them aligned in the same row. The margins are what is creating the space between them.

Step 8: Let's do some box model math, if the container width is 1122px, and each column is 369px wide, that leaves us with 15px for spacing between the columns. So by giving column1 and column2 a margin-right of 7.5px, we use up our space effectively.

Done! With that you have just created a page layout template that can be used to create a website.
