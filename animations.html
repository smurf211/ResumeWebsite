The second argument, input, determines how far we want the bar to grow. Essentially we are modifying the width of the bar. If you notice in the CSS, the width of bar by default is set to 1%. By using an argument of 85, we will grow the bar to 85% width.

The last argument, time determines how quickly the bar grows. An argument of 20 means each frame the bar grows will take 40 ms to process. A second function frame() is used to grow the bar. When the width is greater than the input value, 85 in this case, the function finishes and the move() completes.
