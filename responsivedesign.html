Responsive Design using Media Queries

You never know what screen size your page may be viewed on. Smartphones, tablets, PC's, Smart TV's, Smart watches. Your page has to show its content properly on all of them! Media queries help with this, they allow for the CSS styling of your page to change based on the width and height of the screen it is being view on. As seen below, the columns show three in a row if the browser size is large enough. If you resize the browser, watch how the columns now show vertically.
