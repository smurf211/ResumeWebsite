My Role

I started this venture as a programming intern but my involvement grew into an architect and designer of the application and its functionality moving forward.

Project Difficulties & Solutions

This project required me to learn new software and technologies on the fly. Setting up the environments and simply getting the full-stack application communicating was a big initial hurdle. Using JWT tokens, web hooks, and spring-boot to communicate with, build, and maintain the MS-SQL database was challenging.

Utilizing my knowledge of the AGILE methodology I was able to work with my team to build a powerful architecture for this application. I used realistic goal-setting techniques while pushing myself to learn and implement these technologies successfully.

Features

  • Registration/Login
  • JWT Authentication
  • Custom content generated by rows on the database
  • Smart SQL procedure to dynamically select the best credit card for a given scenario
  • Webhooks for automatic updates on user financial data
